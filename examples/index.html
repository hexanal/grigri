Welcome

grigri is just a bunch of utility classes, mixins, and helpers that help with building a grid system.

How to use

It's super similar to Bootstrap; but it's just about the "grid".

The blocks you see above were built using:

  • a div with a .container class (classic!)
  • a child div with a .row class (easy!)
  • two children divs with .col-6 classes, and then one of them has a .col-lg-4 class to make it smaller on large screens.
    The other one has a .col-lg-6 to fill the remainder space!

We're using the 12-columns system here.

Two files:

grigri.scss

To set up the variables, mixins, etc.

grigri.classes.scss

The list of all the utility classes we can use.

What else?

It's using Sass, because we're making good use of mixins and variables

Responsive

To set it up, we first have breakpoints, here are the default ones:

  • $mq-xs: 500px;
  • $mq-sm: 700px;
  • $mq-md: 1000px;
  • $mq-lg: 1200px;
  • $mq-xl: 1440px;
  • $mq-xxl: 1920px;

Configurable

We're also leveraging the awesome power of CSS variables. Here's what we got:

--max-width: 1200px;
that's the width of the main container column
--narrow-width: 800px;
that's the one used in the container narrow class combo
--wide-width: 1680px;
same as above, but for container wide
--margins: 1rem;
the side margins, around the container (we can ignore those by using the container flush combo)
--gutters: 0.75rem;
spacing between columns, and between items inside a grid class
--spacing: 3.5rem;
unit of vertical spacing
--col-width: calc( (100% - (var(--gutters) * 11) ) / 12);
calculated width of a single column (a.k.a. column unit)

Helpers

What about those?

  • --helper-cols-color: white;
  • --helper-lines-color: magenta;
  • --helper-spacing-color: orangered;
  • --helper-margin-color: lime;
